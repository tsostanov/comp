# comp

Компилятор на Go: лексер + парсер + базовый семантический анализатор.

Семантический анализатор проверяет:
- использование необъявленных переменных;
- использование переменных до инициализации;
- повторное объявление в одном scope;
- неиспользованные переменные.

```bash
go run . program.txt
```
